Leçon : La boucle while

Qu'est-ce qu'une Boucle while ?

La boucle while ("Tant que") répète un bloc de code tant qu'une condition spécifiée est vraie. Elle est parfaite quand on ne sait pas à l'avance combien de tours de boucle seront nécessaires.

Le fonctionnement : 1. On teste la condition. 2. Si elle est vraie, on exécute le code. 3. On retourne à l'étape 1. Si la condition est fausse dès le départ, la boucle ne s'exécute jamais.

Syntaxe de while

int compteur = 0;
while (compteur < 5) { // Tant que compteur est inférieur à 5...
  Serial.println(compteur);
  compteur++; // TRÈS IMPORTANT : modifier la variable de la condition !
}

Piège 1 : La Boucle Infinie

int i = 0;
// ERREUR ! i n'est jamais modifié. La condition (i < 10) sera toujours vraie.
while (i < 10) {
  Serial.println("Au secours, je suis bloqué !");
  // On a oublié d'écrire "i++;"
}

Piège 2 : Quand utiliser while ou for ?

  • Utilisez for quand vous savez combien de fois vous devez répéter (ex: "clignoter 5 fois").
  • Utilisez while quand vous répétez jusqu'à ce qu'un événement se produise (ex: "attendre qu'un bouton soit pressé").
